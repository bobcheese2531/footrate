# README

# アプリケーション名:「Footrate」
 サッカー「football」の試合を観て出場した選手を評価「rate」して、他の人と意見を交換するアプリケーション。  
 
# アプリケーション概要
 世の中にいる全欧州サッカーファンを対象にしたアプリケーションです。  
 欧州サッカーは基本的に深夜にやっていて、見終わった後に誰とも感想を共有することができませんでした。  
 そこでこのアプリを通して意見を交換する場を提供したいです。
